Links externos

a função tecnica da teg "a"

para criar links utilizamos o atributo fundamental "href" (hyper text reference), que indica o destino pra onde o usario será levado. link externo direciona o usuario para outro site, links internos direcionam para outra pagina do seu próprio site

Teg "a"

os links são "a cola" que mantem a internet unida. Sem eles, a web seria apenas uma coleção de documentos isolados em vez de uma red globa conectada. No HTML, os links são criado atraves da teg "a" que significa ancora

Exemplos

  1. links externos
    • Aqui vocês podem acessar as aulas de HTML do professor Janino
  2. links internos
    • Aqui você pode acessar seus proprios sites
