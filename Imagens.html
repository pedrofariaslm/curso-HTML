Imagens

Para colocar uma imagem no HTML, utilize a tag img com o atributo src para indicar o caminho do arquivo e alt para descrição textual (acessibilidade) .A tag é autoencerrável, não precisando de fechamento /img.

Como colocar icones

primeiro passo:

para gerar um icone, localize uma imagem em formato png do tamanho desejado, caso não tenha, utilize o site:

https://www.iconarchive.com

Apos localizar a imagem ou ter feito o download da imagme, acesse o site

https://favicon.io

esse site tem varias opções para gerar um araquivo com a extenção.ICO
feito isso com a imagem, vá no código head, acima do titulo e digite a palavra link. escolha a opção link:fivecon, apague a palavra five.ico e apaerte ctrl + espaço localizando assim o seu icone gerado
