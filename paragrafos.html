Explicação

A hierarquia de títulos em HTML serve para organizar o conteúdo da página de forma estruturada e lógica. Existem seis níveis de títulos, que vão de h1 até h6, sendo h1 o mais importante e h6 o menos importante. O h1 normalmente representa o título principal da página e, em geral, deve ser usado apenas uma vez. Os h2 são usados para dividir o conteúdo em seções principais. Já os h3 representam subseções dentro de um h2, e assim por diante, até o h6, que é o nível mais específico. Essa organização funciona como um índice de livro, onde cada nível representa uma profundidade maior no conteúdo. Por exemplo, um h2 pode ser um capítulo, e um h3 um tópico dentro desse capítulo. É importante seguir a ordem correta dos títulos, sem pular níveis sem necessidade, pois isso melhora a acessibilidade e o SEO da página. Além disso, os títulos devem ser usados para estruturar o conteúdo, e não apenas para definir o tamanho do texto — a aparência deve ser controlada com CSS. Erros comuns incluem usar vários h1 sem necessidade, escolher o título apenas pelo tamanho visual e não manter uma hierarquia lógica.

Paragrafos

Usada para criar blocos de texto. Cada parágrafo representa uma ideia ou informação. O navegador adiciona espaçamento automático entre eles.

Quebra de linha

Serve para quebrar a linha dentro de um mesmo parágrafo, sem criar outro bloco de texto. Deve ser usada com moderação.

Linha horizontal

Cria uma linha que separa conteúdos ou seções da página.

IMPORTANTE

Indica que o texto tem importância. Normalmente aparece em negrito, mas seu principal objetivo é dar significado semântico.

Eenfase

Usada para dar ênfase a uma palavra ou frase. Geralmente aparece em itálico.

Elemento genérico inline

Não tem significado semântico. É usada para aplicar estilos ou manipular pequenas partes do texto com CSS ou JavaScript.

código

Usada para exibir trechos de código dentro do texto, geralmente com uma fonte diferente.
